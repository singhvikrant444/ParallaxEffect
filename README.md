# 🌄 Parallax Effect Website

## 📌 Project Overview

This project demonstrates a **Parallax Scrolling Effect** using HTML and CSS.
Parallax scrolling creates an illusion of depth by moving background images slower than foreground content while scrolling.

It is commonly used in modern websites to create attractive landing pages.

---

## 🚀 Features

* Smooth parallax scrolling effect
* Responsive design
* Clean UI layout
* Pure HTML & CSS implementation
* Beginner-friendly structure
